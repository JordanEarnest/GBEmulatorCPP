# GBEmulatorCPP 🎮
## Description
An emulator for the Game Boy written in C++.
## How to Use
### Linux
1. Clone the repository

```
git clone https://github.com/JordanEarnest/GBEmulatorCPP.git
```

2. Compile

```
make
```

3. Execute

```
./gbemu
```
