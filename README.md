# GBEmulatorCPP
## Description
An emulator for the Game Boy written in C++.
## How to Use
### Linux
1. Clone the repository to your desired directory

```
git clone https://github.com/JordanEarnest/GBEmulatorCPP.git
```

2. Compile using make

```
make
```

3. Execute program

```
./gbemu
```
